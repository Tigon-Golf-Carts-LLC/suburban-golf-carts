# Accessibility Statement - Rehoboth Golf Carts
# Commitment to Digital Accessibility and Inclusive Design

## ACCESSIBILITY COMMITMENT
Rehoboth Golf Carts is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to ensure our website is accessible to all users.

## ACCESSIBILITY STANDARDS COMPLIANCE
Our website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards. These guidelines explain how to make web content more accessible for people with disabilities and user-friendly for everyone.

### Current Compliance Level
- **Target Standard:** WCAG 2.1 AA
- **Testing Method:** Automated and manual accessibility testing
- **Last Audit:** January 21, 2025
- **Next Review:** Quarterly accessibility assessment

## ACCESSIBILITY FEATURES IMPLEMENTED

### Keyboard Navigation
✓ Full keyboard navigation support
✓ Visible focus indicators
✓ Logical tab order throughout the site
✓ Skip links for main content navigation
✓ Keyboard shortcuts for common actions

### Screen Reader Support
✓ Semantic HTML structure for proper screen reader interpretation
✓ ARIA labels and descriptions where appropriate
✓ Alternative text for all meaningful images
✓ Descriptive link text and button labels
✓ Proper heading structure (H1-H6 hierarchy)

### Visual Design Accessibility
✓ High contrast color combinations
✓ Color-blind friendly color palette
✓ Scalable text up to 200% without horizontal scrolling
✓ Clear visual focus indicators
✓ Consistent navigation and layout structure

### Content Accessibility
✓ Plain language usage where possible
✓ Clear and descriptive page titles
✓ Logical content structure and organization
✓ Error messages are clear and helpful
✓ Forms include proper labeling and instructions

### Interactive Elements
✓ Form fields with associated labels
✓ Clear error messaging and validation
✓ Accessible dropdown menus and navigation
✓ Proper button and link identification
✓ Modal dialogs with appropriate focus management

## BUSINESS ACCESSIBILITY SERVICES
Beyond our website, Rehoboth Golf Carts is committed to serving customers with disabilities:

- Accessible vehicle entry and exit assistance

### Service Accessibility
- Large print documentation available upon request
- Verbal explanation of all technical information

### Vehicle Accessibility Options
- Golf cart modifications for accessibility needs
- Wheelchair accessible utility vehicles
- Hand controls and adaptive equipment consultation

## ASSISTIVE TECHNOLOGY COMPATIBILITY
Our website is designed to work with:
- Screen readers (NVDA, JAWS, VoiceOver, TalkBack)
- Voice recognition software
- Keyboard navigation tools
- Browser zoom functions up to 200%
- High contrast and dark mode browsers
- Mobile accessibility features

## ACCESSIBILITY TESTING PROCEDURES

### Automated Testing
- Regular automated accessibility scanning
- WAVE (Web Accessibility Evaluation Tool) testing
- Lighthouse accessibility audits
- Color contrast ratio verification
- HTML validation for semantic structure

### Manual Testing
- Keyboard-only navigation testing
- Screen reader compatibility testing
- Mobile accessibility verification
- User experience testing with accessibility tools
- Focus management and visual indicator testing

### User Testing
- Feedback collection from users with disabilities
- Accessibility consultation with community members
- Regular usability testing with assistive technologies
- Continuous improvement based on user feedback

## KNOWN ACCESSIBILITY LIMITATIONS
We are working to address the following areas:
- Some third-party embedded content may not meet full accessibility standards
- Complex vehicle specification tables may need enhanced screen reader support
